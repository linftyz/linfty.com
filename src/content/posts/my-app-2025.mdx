目前在 Mac 上常用的一些工具软件，目前只想到这些，后面想到了再补充。

- **Ghostty**：日常使用最多的终端，之前用过一段时间的 Alacritty + Zellij 也很不错，日常跑命令、ssh、看日志都在这。
- **Mac Mouse Fix**： 调鼠标滚动方向和速度的，都用的都是默认值。
- **One Switch**：常用系统开关集中到菜单栏，最常用的就是一键保持亮屏，避免进入休眠状态。
- **Snipaste**：截图标注 + 贴图对照，很方便免费的功能已经足够用了。
- **Raycast**：启动、搜索、剪贴板与快捷动作的统一入口。
- **Wins**：窗口分屏/贴边/布局管理，多任务时省时间。
- **Obsidian**：就是日常的记录，用挺久了。
- **Multipass / OrbStack**：需要隔离环境时跑 Linux/容器，用哪个更顺就用哪个。
- **VS Code / Sublime Text**：VS Code 做主力开发；Sublime 用来临时快速查看文件。

等到明年的今天看看有没有啥变化。
